== Description ==
With the help of this plugin, you can change the logo of the WordPress admin section, this plugin makes your work very easy and you can do it with a few clicks.
== Installation ==
This section explains how to install the plugin and set it up.

Enter the wp-content section and then enter the plugins section and upload the file there.
Then enter the plugins section and select the activation option.
